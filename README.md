# Ecommerce-Product-Reviews-Big-Data-Project
Big Data project on E-commerce Product Review Analysis using Hadoop ecosystem tools. Includes dataset, scripts, Pig &amp; Hive queries, and PySpark analytics workflows.


# Big Data Project — E-commerce Product Review Analysis

This repository contains the coursework project for the subject
**Big Data Analytics**, focused on large-scale processing of
e-commerce product reviews using distributed Big Data technologies.

## Project Contents

- Dataset (Customer Product Reviews – CSV)
- Hadoop MapReduce scripts
- Pig & Hive queries
- PySpark processing and analysis
- Project Report (PDF)
- Generated insights & outputs

## Technologies & Tools Used

- Hadoop & HDFS
- MapReduce
- Pig
- Hive
- PySpark
- Distributed Data Processing

## Work Performed

- Processed and cleaned large-scale customer review data
- Performed distributed data transformations
- Executed analytical queries in Pig and Hive
- Implemented PySpark-based review analytics

## Outcomes & Insights

- Extracted product sentiment trends
- Identified customer review patterns
- Derived insights for business decision-making
- Analyzed user preferences and review behavior

## Folder Structure

/data — Raw dataset  
/hadoop — MapReduce jobs  
/pig — Pig scripts  
/hive — Hive queries  
/pyspark — PySpark notebooks & scripts  
/reports — Project documentation  
/results — Output files
